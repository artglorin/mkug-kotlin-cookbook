Взаимодействие с Native кодом

  1. Kotlin код компилируется в нативный код:

    1. в исполняемый файл

    2. в динамические библиотеки

    3. в AppleFramework для работы с Objective-C, Swift

  2. Kotlin код вызывает нативный код

Компиляция Kotlin в Native код

  • Пререквизиты:

    • скачать/собрать kotlin-native compilator

    • загрузить код LLVM

  • Скомпилируем Kotlin код в нативный код, двумя способами:

    • без IDE

    • через Intellij Idea | gradle plugin

Вызов C метода из Kotlin

  • Пререквизиты:
